Defined in: app/coffee/modules/base/service.coffee

Overview

BaseService class definition The BaseService class provides interfaces to fetch models and collections from the server. It is essentially an abstract factory pattern that provides entities when subclassed with the appropriate radioChannel, modelPrototype, and collectionPrototype values
